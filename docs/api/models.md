# Data Models Reference

CRIS uses Pydantic for all data structures to ensure strict validation and easy serialization.

### Case
The primary container for an investigation.
- `id`: Unique UUID/String.
- `title`: Short name of the case.
- `status`: `open`, `closed`, `cold`.
- `crime_type`: `homicide`, `robbery`, etc.
- `priority`: 1-5.

### Person
Represents any human entity in the system.
- `name`: Full name.
- `alias`: List of nicknames.
- `role`: `victim`, `suspect`, `witness`, `poi`.
- `risk_score`: 0-1 (calculated by Profiler).

### Evidence
Represents physical or digital items.

- `extracted_text`: Raw text from OCR or processing.
- `chain_of_custody`: List of custody handlers.
